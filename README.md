# Formal Methods for Software Development (Verification of Functional Programs) - Exercise Solutions
This repository contains the solutions to the exercises of the course "Formal Methods for Software Development" given at Universidad EAFIT by Prof. Andres Sicard Ramirez. The exercises are solved in both Coq and Isabelle/HOL, and the files are organized as follows:
- `coq/`: Contains the Coq solutions to the exercises.
- `isabelle/`: Contains the Isabelle/HOL solutions to the exercises.
- `docs/`: Contains the (Open Source) PDF versions of the exercise sheets.
- `README.md`: This file, which provides an overview of the repository and its contents.

The official course materials, including the exercise sheets and lecture notes, can be found on the course website: [https://asr.github.io/courses/verification-of-functional-programs/2026-1/index.html](https://asr.github.io/courses/verification-of-functional-programs/2026-1/index.html).
